# pApO Project

pApO is an open hardware-inspired connected push-to-talk communication system designed to provide a modern, secure and simple alternative to traditional walkie-talkies.

## Components

- pApOS
- PPP
- pApO Server
- pApO Config
- pApO Simulator
- pApO Flash
- pApO Factory
- pApO Updater

## Repository structure
